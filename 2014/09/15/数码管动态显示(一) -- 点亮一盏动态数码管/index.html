数码管动态显示(一) -- 点亮一盏动态数码管

-动态数码管的特点是将所有静态数码管的段选线并联在一起
-由位选线控制是哪一位数码管有效。

点亮一盏动态数码管

  1. 如何控制某一位数码管
    51单片机默认上电之后所有IO口均为高电平,所以想要控制某一位数码管,就需要先得出8为数码管组成的2进制数,其中0为接通.
    例如:要点亮基数展1,3,5,7,那么所表示的二进制数为01010101,转换为16进制即为0x55.
  2. 如何控制数码管显示特定的数字
    电路图中可以看出数码管位选均并联接地,为共阴极.
    那么同样的点亮即写1.
    一位数码管如图所示分为abcdefg+dp(右下角一点)共8段,
    且a->dp为低位到高位,读数的时候要倒过来.
    比如,想要让数码管亮3,就是abcdg亮,其他全灭.2进制数表示为01001111.
    转换成16进制数便为0×4f.

例程

#include<reg52.h>
#define dula P0
#define wela P1
void main()
{
/*控制某一个数码管亮*/
dula=0xfe;//第一个(电路图中显示为最低位)数码管亮
dula=0x06;//发送段选
while(1);//令程序停止在此处
}
