单片机学习笔记之独立按键

按键开关有4个引脚,视左上为1,顺时针排序.初始情况下1&2(3&4)脚不导通,1&3(2&4)脚导通,所以如果需要自己焊接的情况下只需要用(1’3)或(2’4)脚中的某一脚进行连接.

消抖

按键在闭合和断开时，触点会存在抖动现象,也就是说按下和释放的时候并不是立刻导通,而是会有一定的跳跃,时间大概约为10ms,所以在编写程序的时候需要用到消抖.

程序思路

检测按键是否被按下—->(是)延时消抖—->再次检测按键是否被按下—->(是)—->执行按键后的程序—->松手检测

例程

#include<reg51.h>
#include<intrins.h>    
#define uchar unsigned char
#define uint unsigned int
#define LED P2
sbit K1=P0^0;
void Delay10ms( );   //延时10ms

void main()
{
    uint i,j;
    j=0x01;

    while(1)
    {    
        LED=j;     
        if(K1==0)//检测按键K1是否按下
        {
            Delay10ms();//延时消抖
            if(K1==0)
            {
                j=_cror_(j,1);//1000 0000
                while((i<50)&&(K1==0))     //检测按键是否松开
                {
                    Delay10ms();
                    i++;
                }
                i=0;
            }
        }        

    }                
}

void Delay10ms(void) 
{
uchar a,b,c;
for(c=1;c>0;c--)
for(b=38;b>0;b--)
for(a=130;a>0;a--);
}
